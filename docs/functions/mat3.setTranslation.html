Function setTranslation

  • setTranslation(a, v, dst?): Mat3

    Parameters

    • a: Mat3

      The matrix.

    • v: Vec2

      The vector.

    • Optional dst: Mat3

      matrix to hold result. If not passed a new one is created.

    Returns Mat3

    The matrix with translation set.
